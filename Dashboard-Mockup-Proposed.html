🎨 Design notes for review

This mockup adopts the visual DNA of the /b/{slug} business page — full-bleed hero photography, soft pill chips, layered cards, the AgroYield green as a confident accent, and warm secondary accents (amber for time-sensitive, clay for soft cards). Nothing about the underlying information architecture or routes changes — same nine modules, same nav, same dark mode. Only the surface treatment.

1. Personalised hero strip

Replaces the empty top-of-page with a "good morning" greeting + dynamic stat line + single primary CTA. Tells the user "we know who you are and what changed since yesterday" in one glance.

2. Soft pill stat chips

Five quick stats in pill form below the hero. Each is a one-tap shortcut to the relevant module. Borrowed directly from the contact pills on the business page.

3. Module cards with personality

Same nine modules, but each card now carries a coloured icon block, a "live" badge (12 new, Trending, +47), and a meta footer with one piece of context. Icon flips to filled-green on hover.

4. Hover micro-interaction

Cards lift 3px, gain a subtle shadow, and reveal a green top-border line. Cheap to implement, dramatic perceived-quality lift, particularly for desktop users.

5. Secondary-module strip

Smaller cards for Research / Mentorship / Grants / Learn — quieter but still warm. Uses the clay accent so they don't compete with the primary six.

6. Pattern extends cleanly

Bottom strip shows how the same hero-card composition translates to Opportunities, Marketplace, Price Tracker, and Directory — each with its own warm gradient + relevant editorial moment. One pattern, many surfaces.

⚠️ Pre-Beta scope discipline: only the Dashboard hero strip + softened module cards land before Monday. Everything else (Opportunities/Marketplace/Price/Directory hero treatments) is Week 1 post-Beta. The mockup shows the full vision so we agree on direction; the build is phased.
📷 IP flag: hero photography in this mockup uses Unsplash imagery. Before production, swap to either AgroYield-owned photos, properly licensed stock, or AI-generated imagery — and verify no third-party logos (John Deere, Massey Ferguson, branded chemicals) are visible in any chosen photo.
